จากการปฏิบัติงานใน Lab นี้ ทำให้ผมเข้าใจถึงความสำคัญของการใช้ Semantic HTML เพื่อวางโครงสร้างหน้าเว็บให้สื่อความหมายได้อย่างถูกต้องและอ่านเข้าใจง่ายโดยไม่ต้องพึ่งพา CSS ครับ นอกจากนี้ยังได้พัฒนาทักษะการเลือกใช้แท็กพื้นฐานต่างๆ ให้เหมาะสม ทั้งการจัดลำดับ Heading, การสร้างตารางที่แยกส่วนหัวและเนื้อหาชัดเจน รวมถึงการสร้างฟอร์มรับข้อมูลที่ซับซ้อน ซึ่งถือเป็นพื้นฐานสำคัญในการพัฒนาเว็บที่มีคุณภาพและรองรับการใช้งานจริงครับ เเละสามารถเอาวิธีเขียนของ html ไปต่อยอดในการเขียนภาษาอื่นๆได้ โดยการจัดเรียงโค้ดให้สวยงามง่ายต่อการอ่าน ทำให้การเขียนโค้ดเป็นระบบมากขึ้น
